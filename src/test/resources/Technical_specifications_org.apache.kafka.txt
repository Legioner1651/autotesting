на Java, используя библиотеки org.apache.kafka создать продюсера,
передающего сообщения с типом key = тип Long, value = с типом JSON.
нужно создать отдельные классы:
1) SendMessages - отправляет key и value произвольного типа данных
2) EqSD - класс создающий объект с типом JSON
3) KafkaTest - тестовый класс который передает сообщение в топик: testing-topic
